1. Obstacle Avoiding Bot
An autonomous robot designed to navigate through environments by detecting and avoiding obstacles in its path.
Microcontroller: Arduino Uno
Sensor: HC-SR04 Ultrasonic Sensor
Actuators: DC Motors with L293D Motor Driver
Logic: The bot continuously measures the distance to objects. If an obstacle is detected within 20cm, it reverses or turns to find a clear path.
2. Smart Traffic Light System
A simulation of a multi-way traffic signal with timed transitions and pedestrian crossing logic.
Components: LEDs (Red, Yellow, Green), Resistors, Pushbutton.
Logic: Uses delay() or millis() based timing to cycle through signals.
